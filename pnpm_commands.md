# pnpm Commands

## Install pnpm

```bash
npm install -g pnpm
```

## Initialize

```bash
pnpm init
```

## Install package

```bash
pnpm add axios
```

## Install dev dependency

```bash
pnpm add -D nodemon
```

## Remove package

```bash
pnpm remove axios
```

## Install dependencies

```bash
pnpm install
```
